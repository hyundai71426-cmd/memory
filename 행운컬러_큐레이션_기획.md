# 오늘의 행운 컬러 — 기획

> 메인 공유 후크. 애드센스 수익을 보완하는 쿠팡파트너스 딥링크 연결 지점.
> 항상 제공(게이트 없음), 타로 결과 화면과 함께 한 번에 공개.

---

## 1. 역할 및 위치

- **결과 퍼널 3번째 단계** — 오늘의 카드(워밍업) → 운세+타로(최종 점수 한 번에 공개) → **오늘의 행운 컬러(메인)** → 행운 넘버(보너스) → …
- 타로 3장 결과 화면에서 점수·해설과 **함께 한 화면에** 노출. 별도 단계 이동 없음.
- 결과 카드 이미지(9:16 / 정사각형)에 색상 블록 + 이름 포함 → SNS 공유 시 자연스럽게 노출.

## 3. 색상 결정 로직

### 기본 원칙
- 생년월일 + 오늘 날짜를 시드로 한 **결정론적 랜덤** → 같은 사람이 같은 날 다시 봐도 동일한 색상.
- 타로 카드는 보정에 영향을 주지 않음(복잡도 대비 효과 낮음).

### 산출 방식

```
색상_인덱스 = hash(생년월일 + 오늘날짜_YYYY-MM-DD) mod 24
```

- `hash()`: 양쪽 입력을 합산하고 단순한 정수 해시(예: 각 자릿수 합산 또는 dayOfYear + birthdayNumber 조합).
- 구체 구현은 기획안 4.1~4.2의 베이스 점수 시드 함수를 재사용 — 같은 사람, 같은 날이면 색상도 일관성 있게 유지.

## 4. UI 표현

### 결과 화면 내 배치
```
┌────────────────────────────────┐
│  오늘의 행운 컬러              │
│  ●  로즈쿼츠                   │  ← 색상 원(48px) + 이름
│  #f4a7b5                       │  ← 헥스 코드(서브 텍스트, 작게)
│                                │
│  [ 쿠팡에서 이 컬러 상품 보기 ] │  ← 쿠팡 딥링크 버튼
└────────────────────────────────┘
```

- 색상 원: 지름 48px, 그림자 효과(은은하게), 색상 배경 원 1개.
- 이름: 16px 세미볼드, 색상 원 오른쪽.
- 헥스 코드: 12px 그레이, 소소한 디테일 — "공유하고 싶은 정보" 역할.
- 버튼 문구: "이 컬러 쇼핑하기" 또는 "쿠팡에서 보기".
- 버튼 위치: 행운 넘버 바로 위, 금전운 점수 영역 아래(섹션 구분선으로 분리).

### 결과 카드 이미지(공유 이미지) 반영
- 9:16 카드 중단부에 색상 가로 배너(상단 16px 높이 스트라이프 또는 하단 색상 블록)로 배치.
- 카드 내 텍스트: "오늘의 행운 컬러 · 로즈쿼츠 #f4a7b5" 한 줄.
- 공유 이미지에서 헥스 코드가 보여야 "이 색 뭐야?" 반응을 유도할 수 있음.

## 5. 쿠팡파트너스 딥링크 구성

### URL 구조
```
https://link.coupang.com/a/{파트너ID}?url=https%3A%2F%2Fwww.coupang.com%2Fnp%2Fsearch%3Fq%3D{검색어_URL인코딩}
```

- `{파트너ID}`: 쿠팡파트너스 가입 후 발급받는 고유 ID.
- `{검색어}`: 색상 풀 표의 "쿠팡 검색 키워드" 중 하나를 랜덤 또는 카테고리 순서대로 사용.

### 검색어 선택
- 각 색상별로 키워드 2개를 미리 정의(위 표 참고).
- v1에서는 첫 번째 키워드 고정 사용(단순하게). v2 이후 GA 클릭률 보고 키워드 교체.

### 구현 시 주의
- 쿠팡파트너스 가입 및 파트너 ID 발급은 **코딩 전에 선행** 필요(가입 심사 최대 3~5일 소요).
- 링크 생성은 프론트엔드에서 직접 조립(서버 불필요). 단, 파트너 ID는 환경변수로 관리해 소스코드에 하드코딩 금지.

## 6. 애드센스 정책 준수

- 쿠팡 딥링크 버튼은 **광고 단위와 혼동되지 않도록** 명확하게 "쿠팡 제휴 링크" 또는 "파트너스 링크" 표기 권장(일부 정책 해석 기준).
- 결과 화면의 광고 슬롯(배너)과 쿠팡 버튼이 너무 가깝지 않도록 섹션 간격 확보.
- 쿠팡파트너스 자체는 애드센스 정책 위반이 아니나, 버튼 배치가 "광고처럼 보이는 위치"(기획안 5번 금지 구역)에 해당하지 않는지 확인 필요.
- `모바일UX_광고전략_기획.md`의 광고 밀도 가드레일 적용: 쿠팡 버튼이 있는 섹션 근처에 광고 단위 추가 삽입 금지.

## 8. 미결 사항

1. **쿠팡파트너스 파트너 ID** — 코딩 시작 전에 가입/발급 완료 필요(선행 과제).
2. **버튼 문구 확정** — "이 컬러 쇼핑하기" vs "쿠팡에서 이 컬러 보기" vs "행운 컬러 상품 구경하기" — UX 카피 최종 선택 필요(안은 `UX카피_고정문구.md`에 추가 권장).
3. **제휴 링크 표기 문구** — "(쿠팡파트너스 제휴 링크)" 표기 위치 및 크기 결정 필요.
