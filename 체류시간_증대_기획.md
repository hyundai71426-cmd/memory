# 체류시간 증대 기획

하루 1회 고정 정책(4.3) 때문에 "재시도"로 체류시간을 늘릴 수 없으므로, 한 번의 방문 안에서 더 많은 걸 보고 머물게 하는 3가지 장치를 설계한다.

## 1. 78장 카드 둘러보기 갤러리 (`/tarot/`)

- 기존 사이트맵의 `/tarot/` 목록 페이지를 단순 리스트가 아닌 **카드 뒤집기 갤러리**로 구현. 78장이 카드 뒷면(7번 비주얼 컨셉의 별/문양 패턴)으로 깔려 있고, 클릭하면 그 자리에서 앞면+짧은 의미가 뒤집히며 공개.
- 오늘 이미 뽑은 카드와 무관하게 **언제든 몇 장이든** 뒤집어볼 수 있음(점수/행운 넘버에는 영향 없음, 순수 탐색 콘텐츠).
- 메이저/마이너 슈트별 필터(완드/컵/소드/펜타클/메이저) 추가해 탐색 경로 다양화.
- 카드 클릭 시 `/tarot/[card-name]` 상세 페이지로 이동하는 링크도 함께 제공해 콘텐츠 페이지 체류로 자연스럽게 연결.

## 2. 결과 화면 연쇄 유도 (퍼널)

각 기능의 결과 화면 하단에 "다음으로 해볼 거리"를 항상 1~2개 제시해, 한 세션 안에서 여러 기능을 순서대로 타도록 유도한다.

| 결과 화면 | 다음 CTA |
|---|---|
| 오늘의 운세 점수 | "타로 3카드도 뽑아보기" |
| 타로 3카드 결과 | 금전운 70점 이상 → "행운 넘버 보기" / 70점 미달 → "오늘의 카드로 운명 비틀기" |
| 행운 넘버 / 운명 비틀기 결과 | "오늘의 TOP 5 랭킹 확인하기" (4.7) |
| 오늘의 랭킹 | "애인/친구와 궁합 모드 해보기" |
| 궁합 모드 결과 | "78장 카드 갤러리 둘러보기" (위 1번) 또는 관련 별자리/띠 페이지 링크 |

- 마지막 단계(갤러리)는 무한 탐색이 가능한 콘텐츠라 자연스럽게 체류시간을 흡수하는 종착점 역할.
- 각 CTA는 "건너뛰기"도 항상 가능해야 함 — 강제 퍼널이 아니라 제안 수준으로 설계(이탈 방지보다 자연스러움 우선).

## 3. 무제한 플레이 사이드 콘텐츠 — "나의 타로 성향 테스트"

- 생년월일 기반 운세와 완전히 분리된 **별도 미니 퀴즈**. 점수/행운 넘버에 영향 없고, 날짜 잠금도 없어 하루에 몇 번이든 재플레이 가능.
- 형식: 5~7개의 가벼운 성향 질문(예: "스트레스 받을 때 나는?") → 답변 조합에 따라 22장 메이저 아르카나 중 1장이 "당신을 닮은 카드"로 매칭.
- 결과 화면에 해당 카드 설명(`/tarot/[card-name]` 링크) + "친구에게 공유하기"/"다시 테스트하기" 버튼.
- 페이지: `/quiz/tarot-type` (또는 메인 내 별도 섹션). 사이트맵에 추가 필요.
- 운세 텍스트와 달리 **고정 매칭 로직**(질문-답변 조합 → 카드 매핑 테이블)이라 AI 호출 비용 없음 — 운영비 부담 없는 콘텐츠.
- 추후 시즌마다 새 퀴즈 추가 가능(예: "나의 2026 행운 동물은?" 등) — 3순위 로드맵의 "시즌 한정" 컨셉과도 연결 가능.

## 4. 기획안 반영 필요 사항

- 사이트맵에 `/quiz/tarot-type` 추가.
- 3.2(타로카드 페이지) 설명에 갤러리형 `/tarot/` 목록 페이지 디테일 보강.
- 4번 핵심 로직에 4.8(결과 화면 연쇄 유도) 서브섹션 추가.
- 8번 로드맵에 "무제한 플레이 사이드 콘텐츠(타로 성향 테스트)" 신규 항목 추가.
