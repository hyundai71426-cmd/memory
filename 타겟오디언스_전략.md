# 타겟 오디언스 전략 — 2030 여성 / SNS 공유 민감군

기획안.md 1번(개요)에 추가되는 타겟 정의와, 이로 인해 기존 기능 설계에 미치는 영향 정리.

## 1. 타겟 정의

- **핵심 타겟**: 20~30대 여성, SNS(특히 인스타그램) 사용이 활발하고 결과를 캡처/스토리 공유하는 데 익숙한 사용자.
- 실제로 인스타그램 이용 성장은 2030·여성이 주도하는 추세이고, 타로/사주 앱들도 궁합·행운 번호 같은 공유형 콘텐츠를 핵심 기능으로 두는 경우가 많음 — 이 사이트가 이미 설계한 궁합 모드, 오늘의 랭킹, 결과 카드 저장 기능과 타겟이 잘 맞아떨어짐.

## 2. 기존 기능과의 정합성 — 그대로 가도 되는 것

- **미스틱 인디고 비주얼 컨셉**(7번): 다크+골드 톤의 "미스틱/위치" 감성은 최근 2030 여성 사이에서 선호되는 무드와 맞음. 비주얼 방향 수정 불필요.
- **궁합 모드(4.6)**: 연인/짝사랑/친구와 공유하기 좋은 전형적인 SNS 공유형 콘텐츠. 타겟에 가장 강하게 부합 — 우선순위를 더 끌어올릴 만함.
- **오늘의 랭킹(4.7)**: 이니셜 노출 + 경쟁 요소는 "내 친구한테 자랑하기" 공유 동기를 자연스럽게 만듦.
- **타로 성향 테스트(`체류시간_증대_기획.md`)**: "나는 어떤 카드?" 식의 유형 테스트는 2030 여성 사이에서 잘 퍼지는 콘텐츠 포맷(MBTI 테스트류와 동일 메커니즘).

## 3. 타겟에 맞춰 보강이 필요한 부분

### 3.1 결과 카드 저장 이미지 — 인스타 스토리 비율 최적화
- 기존 "결과 카드 저장하기"(3.1) 이미지를 **9:16 세로형(인스타 스토리 사이즈)**을 기본값으로 제공. 정사각형(피드용) 버전도 옵션으로 추가.
- 이미지 안에 작게 "@사이트명" 또는 URL 워터마크를 넣어, 스토리로 공유됐을 때 출처가 보이게(8번 로드맵의 QR코드 아이디어와 같은 목적, 텍스트 버전).
- **워터마크 URL에 추적 파라미터 부착(확정)**: 그냥 메인 URL이 아니라 `?utm_source=share&utm_medium=card&utm_campaign=<기능명>` 형태로 기능별로 구분(예: 궁합 모드 카드는 `utm_campaign=compatibility`, 오늘의 운세 카드는 `utm_campaign=daily`). 분석 도구(GA4 등)에서 소스별 유입을 바로 구분할 수 있게 함 — 비용 거의 없이 5번 측정 항목의 핵심 데이터 소스가 됨.

### 3.2 톤앤매너 — 카피 가이드 보강
- 기존 `운세텍스트_생성방식.md`의 "친근한 존댓말" 기조를 유지하되, 2030 여성향 트렌디 표현을 가볍게 허용(예: "오늘 운세 찐맛집", "이 카드 좀 소름" 같은 자연스러운 구어체). 단, 과도한 신조어/유행어 남발은 지양 — 톤이 빨리 촌스러워질 수 있으므로 "친근하지만 과하지 않게"가 기준.
- 광고/콘텐츠 정책상 단정적 문구 금지 규칙(`운세텍스트_생성방식.md` 5번)은 동일하게 유지.

### 3.3 공유 시 친구 태그 유도 문구
- 궁합 모드, 오늘의 카드 결과 화면에 "이 결과 캡처해서 친구한테 보내보기" 같은 가벼운 공유 유도 문구 추가(강제 X, 자연스러운 제안).

### 3.4 콘텐츠 페이지 어조도 동일 기조로
- 별자리/띠 페이지(`별자리띠_콘텐츠기획.md`)도 정보 전달은 정확하게 하되, 예시·비유를 2030 여성이 공감할 수 있는 상황(연애, 직장 관계, 친구 관계 등)으로 우선 구성.

## 4. 측정 및 성공 기준 (바이럴 가정 검증)

이 전략 전체가 "SNS 공유가 잘 될 것"이라는 검증되지 않은 가정 위에 있음. 출시 후 가정이 맞는지 확인할 수 있도록 최소한의 측정 체계를 둔다.

- **분석 도구**: GA4(또는 동급 무료 분석 도구) 도입을 1차 출시 범위에 포함.
- **추적할 이벤트**: "결과 카드 저장" 버튼 클릭, 공유 유도 문구 클릭(`3.3`) — 둘 다 클라이언트 이벤트로 간단히 남김.
- **핵심 성공 지표 (1차 기준값, 실측 후 조정)**:
  - 저장 버튼 클릭률(결과를 본 사람 중 저장한 비율) — 1차 기준 10% 이상이면 "공유 동기는 작동 중"으로 판단.
  - 워터마크 UTM 유입(위 3.1) — 이 값이 전체 신규 방문의 일정 비율(예: 5% 이상) 이상이면 "실제로 공유→유입 루프가 작동"으로 판단.
- **판단 시점**: 출시 후 2~4주 데이터를 보고 위 두 지표가 기준 미달이면 바이럴 전략을 재검토(카피/유도 문구 수정 또는 우선순위 하향).

## 5. 바이럴 실패 시 플랜B — SEO 콘텐츠를 백업 채널로 명시

- 바이럴 루프가 기대만큼 안 돌더라도, 별자리/띠/가이드 콘텐츠 페이지(`기획안.md` 3.3, 3.4)는 애드센스 운영을 위한 부산물이 아니라 **검색 유입이라는 독립적인 트래픽 채널**로 명시적으로 취급한다.
- 즉 콘텐츠 페이지의 SEO 품질(키워드, 내부링크 허브 역할)은 바이럴 성공 여부와 무관하게 1차 출시부터 신경 써야 하는 항목으로 우선순위를 유지 — "공유가 잘 안 되면 콘텐츠 SEO가 메인 성장 채널이 된다"는 전제를 명확히 함.

## 6. 우선순위 재조정 제안

- 기존 로드맵(8번)의 2순위 항목 중 **궁합 모드·오늘의 랭킹**은 이 타겟 전략상 우선순위를 가장 높게 유지.
- 3순위의 "히스토리 기록"보다 SNS 공유형 콘텐츠(타로 성향 테스트, 9:16 공유 이미지)를 먼저 다듬는 쪽이 타겟 적합도가 높음 — 로드맵 순서 자체를 바꾸진 않되, 같은 순위 안에서는 공유형 기능을 먼저 다듭는 걸 권장.
