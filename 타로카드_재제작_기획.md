# 타로카드 78장 재제작 기획

## 1. 배경

기존 방식(파이썬 코드로 원본 RWS1909 스캔본 복원·보정)은 카드마다 결과물이 제각각이라 폐기. 코딩으로 자동 보정하는 대신, 78장을 하나씩 손으로 새로 그려 브랜드 톤(미스틱 인디고)을 통일하는 방향으로 전환.

**원칙: 창작이 아니라 재현.** 인물 배치, 들고 있는 사물, 숫자·슈트 구성 등 원작 RWS1909의 구도와 상징은 그대로 따라가고, 화풍·색감·선 처리만 새로 입힌다. 타로 상징 체계를 바꾸면 카드 의미 해석 콘텐츠(`타로_정역방향_한줄해설.md`, `타로카드_상세페이지_본문.md`)와 어긋나므로 구도 변경은 금지.

## 2. 아트 스타일 가이드

기존에 승인된 이미지 에셋(`이미지에셋_시안_v1/`)과 동일한 언어를 그대로 가져온다.

- **캔버스**: 300×520 viewBox (카드뒷면 시안과 동일 비율)
- **배경색**: `#211b3d` (딥 인디고)
- **프레임**: 카드뒷면과 동일한 이중 프레임 — 외곽 골드 2px(`#c9a227`) + 내곽 라이트인디고 1px(`#4a3f7a`), 모서리 곡선 장식 동일하게 재사용. 78장 전부 같은 프레임을 쓰면 세트로 봤을 때 통일감이 생기고, 프레임은 모든 카드에 공통이라 한 번만 만들어 재사용 가능.
- **카드 번호/이름 표기**: 프레임 하단에 작은 골드 텍스트로 카드명(국문) — 카드뒷면에는 없던 요소지만 상세페이지·결과카드에서 카드 식별용으로 필요.
- **삽화 디테일 레벨**: 띠 동물 일러스트(단순 실루엣)보다는 한 단계 디테일을 올린다. 원작에 있는 핵심 요소(인물 동작, 주요 사물 2~4개, 배경 요소 1~2개)는 살리되, 잔무늬·그림자·텍스처는 다 덜어내고 평면 채색 + 골드 윤곽선으로 단순화. 즉 "정밀 모사"가 아니라 "원작 구도를 따라간 플랫 벡터 아이콘".
- **팔레트 제한**: 인디고 계열 2~3톤(`#211b3d`, `#2c2456`, `#4a3f7a`) + 골드 계열 1~2톤(`#c9a227`, 밝은 골드 하이라이트) + 크림색 1톤(인물 피부/하이라이트용, 과도한 채도 금지. 78장 전체가 이 팔레트 밖으로 나가지 않아야 "카드마다 제각각" 문제가 재발하지 않음.
- **선 굵기**: 주요 윤곽선 1.4~1.6px, 세부선 0.8~1.2px로 두 단계만 사용 (지금 아이콘 시안 기준과 동일).

## 3. 메이저 아르카나(22장) vs 마이너 아르카나(56장)

- **메이저 22장**: 카드마다 구도가 전부 다름 — 노출 빈도가 가장 높고(오늘의 카드, OG 이미지, 마케팅) 상징성도 강하므로 한 장씩 개별 설계. 우선 제작 대상.
- **마이너 56장(완드/컵/소드/펜타클 × 14)**: 슈트별로 반복되는 패턴이 있어 모듈화 가능 (아래 4번).

## 4. 마이너 아르카나 모듈화 전략

56장을 전부 백지에서 그리면 메이저 22장보다 일이 더 커진다. 생산 속도를 위해 슈트별 공통 모듈을 먼저 만든다.

- **슈트 심볼 1개씩 확정**: 완드(지팡이)·컵(성배)·소드(검)·펜타클(별 모양 동전) 각 1개 아이콘을 원작 형태 기준으로 골드 라인으로 디자인 — 4종만 만들면 끝.
- **핍 카드(에이스~10, 숫자카드 4슈트×10장=40장)**: 원작 RWS는 숫자카드도 단순 사물 배열이 아니라 작은 장면(인물·배경 포함)이 그려져 있음 — 이 구도는 유지하되, 배경 디테일은 가장 단순화해서 슈트 심볼 개수/배치 + 핵심 인물 동작만 표현. 카드 10장씩은 "심볼 개수가 늘어나는" 동일 틀에 인물 포즈만 바뀌는 식으로 모듈 재사용.
- **코트카드(시종·기사·여왕·왕 × 4슈트=16장)**: 인물 일러스트가 핵심이므로 핍 카드보다 디테일을 한 단계 더 줌. 슈트별 색 포인트(완드=주황빛 골드, 컵=청록빛 인디고 등 미세 톤차)는 두지 않고 슈트 심볼만 바꿔서 통일성 유지 — "카드마다 제각각" 재발 방지.

## 5. 제작 순서 및 파일럿

1. **파일럿 1장**: The Fool(0번) — 메이저 중 가장 단순한 구도라 스타일 가이드 검증용으로 적합. 이 한 장으로 프레임·팔레트·디테일 레벨 확정.
2. **파일럿 승인 후**: 메이저 22장 순서대로 제작 (0~21).
3. **마이너 모듈 확정**: 슈트 심볼 4종 + 핍/코트 템플릿 1세트씩 제작 → 승인.
4. **마이너 56장 일괄 제작**: 확정된 모듈로 숫자만 바꿔가며 생산.
5. 각 단계마다 결과를 바로 승인받고 다음 단계로 — 한 번에 78장을 다 그려놓고 "역시 별로였다"가 되는 걸 막기 위함.

## 6. 파일 규격·네이밍

- 포맷: SVG (벡터, 카드뒷면·아이콘 시안과 동일)
- 캔버스: 300×520 viewBox
- 네이밍: 기존 `tarot_images/RWS1909_-_00_Fool.jpeg` 패턴을 참고해 `tarot_remastered/00_Fool.svg`, `01_Magician.svg` … `Wands_01.svg` 등 일관 규칙 사용 (메이저는 번호_영문명, 마이너는 슈트명_숫자)
- 보관 위치: `이미지에셋_시안_v1/` 옆에 `타로카드_재제작_v1/` 신규 폴더

## 7. 일관성 검수 절차

- 매 5~10장 완성 시점마다 78장(또는 진행분)을 하나의 그리드 이미지로 모아 축소 비교 — 팔레트·선 굵기·프레임 위치가 카드마다 어긋나지 않는지 한눈에 확인.
- 메이저 22장 완료 시 1차 전체 검수, 마이너 56장 완료 시 2차 전체 검수.
- 검수 기준: (1) 원작 구도·상징 보존 여부, (2) 팔레트가 가이드 색상 밖으로 나갔는지, (3) 프레임/카드명 표기 위치가 78장 전부 동일한지.

## 8. 일정 감

- 파일럿 1장: 즉시 확인 가능한 분량
- 메이저 22장: 파일럿 스타일이 확정되면 가장 시간이 걸리는 구간 (장당 구도가 다 다름)
- 마이너 56장: 모듈화 덕분에 메이저보다 장당 작업 속도가 빨라야 함
