# 콘텐츠 세부작성 투두 리스트

기획 문서들은 거의 다 완성됨. 이제부터는 "실제 텍스트/데이터/이미지"를 채우는 단계. 코딩 착수 전 콘텐츠만 모아 정리.

## 1. 타로 78장 — 카드별 콘텐츠

- [x] **정/역방향 한 줄 해설 (폴백용)** — 78장 × 정/역 2개 = 156개 문구. `운세텍스트_생성방식.md` 3번에서 예고된 작업. `타로_정역방향_한줄해설.md`로 분리 작성 완료.
- [x] **카드 상세 페이지 본문** (`/tarot/[card-name]`, 78개) — 상징/정방향/역방향/연애·직업·금전 키워드, 500~700자. `타로카드_상세페이지_본문.md`로 작성 완료.
- [x] **78장 갤러리용 짧은 의미 한 줄** — 기존 `타로_정역방향_한줄해설.md`를 그대로 재사용하기로 결정(신규 작성 불필요).

현재 `tarot_images/` 51/78장만 확보됨.

- [x] **완드(Wands) 14장 전체 누락** — 위키미디어 커먼즈(Roses & Lilies 판)에서 소싱 완료
- [x] **소드(Swords) 13장 누락** — 위키미디어 커먼즈에서 소싱 완료

## 3. 별자리·띠 콘텐츠 페이지 (24개) — 본문 미작성

`별자리띠_기초데이터.md`(날짜/원소/오행/궁합표)는 완성. 본문 텍스트 자체는 아직 없음.

- [x] 별자리 12개 본문 — `별자리_콘텐츠_초안.md`로 작성 완료(AI 초안 단계, 사람 검수 전)
- [x] 띠 12개 본문 — `띠_콘텐츠_초안.md`로 작성 완료(AI 초안 단계, 사람 검수 전)
- [x] `별자리띠_콘텐츠기획.md` 5번 방식대로 Claude 1차 검수 완료 — 기초데이터 대비 사실 오류 없음 확인, 중복 비유 문장 2건(게자리↔전갈자리, 토끼띠/양띠→돼지띠) 발견 후 차별화 수정. 사람 최종 확인 전 단계.

## 4. 운세 텍스트 폴백 템플릿 풀

`운세텍스트_생성방식.md` 3번에서 정의된 구조, 실제 문장은 미작성.

- [x] 점수 구간 5개(0~20/21~40/41~60/61~80/81~100) × 카테고리 5개(종합/금전/애정/직장/건강) × 문장 4개 = 100개 문구. `운세_폴백_템플릿.md`로 작성 완료.
- [x] 톤 가이드 준수 확인(친근한 존댓말, 단정적/공포 조성 문구 금지, 트렌디 표현은 과하지 않게) — 파일 내 "비고" 섹션에 점검 결과 기록.

## 5. 궁합 모드 — 관계유형별 해석 콘텐츠

`관계궁합_확장_기획.md` 기준, 4유형(연인/베프/상사/짝사랑) 콘텐츠 분화 필요.

- [x] 유형별 톤에 맞춘 해석 문구(또는 AI 프롬프트 가드레일) — 특히 베프궁합은 "연애" 표현을 "케미/우정"으로 치환하는 어휘 매핑표 필요. `궁합_관계유형별_해석가이드.md`로 작성 완료(베프/상사 매핑표+가드레일 포함).
- [x] 짝사랑궁합 전용 안전장치 문구(단정적 결론 금지 가드레일) 구체화 — 위 파일 4번 섹션에 금지표현/대체표현/AI 가드레일/고정 안내문구까지 포함.
