# 네이버 콘텐츠 작성 규칙 (2025~2026 최신)

---

## 1. 검색 알고리즘 구조

### 스마트블록 (2025년~ VIEW 완전 대체)
네이버 검색 결과는 단순 나열이 아닌 주제별·의도별 블록 단위로 재구성된다.

| 블록 유형 | 노출 로직 |
|---|---|
| 인기 주제 | 체류시간·공감·댓글 등 반응 지표 종합 |
| 내또래 | 검색자와 비슷한 연령·성별이 많이 본 글 |
| 함께 찾은 콘텐츠 | 연관 검색 여정 기반 시리즈 글 추천 |
| AI 추천 | 하이퍼클로바X가 문서 맥락·정확성·전문성 평가 |
| VIEW 통합 | 블로그·카페·포스트를 품질순으로 정렬 |
| 지식 스니펫 | 질문형 검색에 직접 답변 요약 표시 |

### 3C 평가 기준
- **Context** — 문서가 검색 키워드의 의도에 얼마나 집중하는지
- **Content** — 전문성, 문서량, 신뢰 출처 등 전반적 품질
- **Chain** — 사용자 상호작용 및 참여 지표 (체류시간, 공감, 댓글)

### Next N Search (2025년 말~ 순차 롤아웃)
- **AuthGR** — LLM 기반 신뢰도 평가 AI. 조회수 중심이 아닌 "누가 썼는지(작성자 전문성·출처)" 중심으로 평가
- **QUMA-VL** — 비전-언어 모델. 텍스트와 이미지의 일관성을 분석해 불일치 콘텐츠에 패널티 부여
- **RCON** — 검색 쿼리의 맥락을 추론해 결과를 재구성하는 랭킹 알고리즘
- **AI 브리핑** — 검색 의도를 요약해 상단에 직접 노출. 인용된 글에 유입 효과 발생

## 2. 콘텐츠 작성 핵심 원칙

### ① 경험 기반 원본 콘텐츠
- AI 자동 생성 글, 광고성 도배 글은 실시간 스팸 필터(AuthGR)로 차단
- 직접 경험·실거래·현장 정보 등 1차 정보가 포함된 글이 우선 노출
- 다른 블로그를 그대로 복붙하거나 재구성한 2차 콘텐츠는 불리

### ② AI 인용 가능한 구조 (AEO)
- 각 H2 섹션의 **첫 문장은 정의형 문장**으로 시작
  - 예: "역삼동 사무실 임대료는 보증금 1,000~3,000만 원, 월세 100~250만 원 수준이다."
- 같은 섹션 안에 **최소 2개 이상의 구체적 수치** 포함
- 수치에는 반드시 **출처 또는 기준 시점** 명시
- 이 구조를 적용하면 AI 요약 블록(지식 스니펫, AI 브리핑)에 인용될 확률이 높아짐

### ③ 이미지와 텍스트 일관성
- 본문 주제와 무관한 이미지 삽입 시 QUMA-VL이 패널티 부여
- 이미지 alt 텍스트는 본문 키워드와 연관성 있게 작성
- 이미지 1장 이상 필수 (주제와 직접 관련된 실사 이미지 권장)

### ④ 사용자 체류시간·반응 지표 관리
- 체류시간이 길수록 알고리즘이 "유용한 콘텐츠"로 판단
- 공감, 댓글, 저장 등 사용자 반응이 인기 주제 블록 노출에 영향
- 독자가 끝까지 읽도록 중간에 소제목, 표, 리스트 등으로 구조화

### ⑤ 주제 전문성 및 블로그 활동성
- 특정 주제에 꾸준히 포스팅할수록 해당 분야 전문 블로그로 인정
- 주제 분산 없이 카테고리를 좁게 유지하는 것이 유리
- 꾸준한 발행 주기가 블로그 활성도 신호로 반영

### ⑥ 콘텐츠 포맷을 주제에 맞게
- 정보성 글: 텍스트 중심 + 표·리스트 활용
- 장소·리뷰 글: 실사 이미지 다수 + 장소 태그 필수
- 질문형 키워드: Q&A 블록 삽입 (지식 스니펫 노출 목표)

## 3. 제목 작성 원칙

- 핵심 키워드를 제목 앞쪽에 배치
- 구체적 수치·조건 포함 (예: "역삼동 사무실임대 20평 보증금 2천")
- 클릭 유도 낚시성 제목보다 정보 전달 중심
- 검색자가 실제 입력할 법한 자연어 형태 권장

---

## 4. 본문 구조 원칙

1. **도입부** — 핵심 정보 요약 (검색 의도에 즉시 답하는 정의형 문장)
2. **상세 섹션** — 소제목별로 분리, 섹션당 200자 이상이면 추가 분할
3. **Q&A 블록** — 예상 질문 2~3개 + 구체적 수치 포함 답변
4. **마무리** — 핵심 요약 또는 행동 유도 (과도한 CTA 금지)

## 5. 절대 금지 사항

| 금지 항목 | 이유 |
|---|---|
| AI 자동 생성 글 그대로 발행 | AuthGR 스팸 필터 차단 |
| 키워드 단순 반복 나열 | 품질 페널티 |
| 텍스트와 무관한 이미지 삽입 | QUMA-VL 페널티 |
| 근거 없는 수치·출처 불명 정보 | 신뢰도 하락 |
| 타 블로그 복붙 또는 재구성 | 중복 콘텐츠 처리 |
| 과도한 광고·제휴 링크 도배 | 광고성 블로그 분류 |
