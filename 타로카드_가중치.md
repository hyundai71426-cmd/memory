# 타로 78장 카테고리별 가중치 테이블

기획안.md 4.2(점수 산출) 로직에서 사용하는 카드별 가중치 원본 데이터.

## 산정 방식

- 카테고리: 금전 / 애정 / 직장 / 건강
- 점수 범위: **-10 ~ +10** (정방향 기준)
- **역방향이면 모든 카테고리 점수의 부호를 반전**한다 (기획안 4.2 규칙)
- 종합 점수는 4개 카테고리 평균으로 계산
- 최종 적용: `최종 점수 = 베이스 점수 + 가중치` (카테고리별, 0~100 클램핑)

## 비고

- 위 점수는 전통적 RWS 카드 의미(상징·키워드)를 기반으로 한 1차 초안이며, 실제 서비스 운영 후 사용자 반응이나 점수 분포(너무 한쪽으로 몰리는지 등)를 보고 조정 가능.
- 종합 점수가 필요한 화면에서는 4개 카테고리 평균을 사용하거나, 필요시 카테고리별 가중치를 다르게 줄 수 있음(예: 메인 화면 "오늘의 총운"은 평균, 개별 카테고리 카드는 해당 값 그대로).
