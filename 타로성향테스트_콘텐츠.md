# 나의 타로 성향 테스트 — 콘텐츠 설계 (`/quiz/tarot-type`)

`체류시간_증대_기획.md` 3번에서 정의된 무제한 플레이 사이드 콘텐츠의 실제 질문/매핑/결과 문구. 생년월일 기반 운세와 완전히 분리된 고정 로직(AI 호출 없음)이라 운영비 부담이 없다.

## 1. 매칭 로직 설계

질문 6개에 답하면 답변들의 "성향 벡터"를 평균 내어, 22장의 메이저 아르카나 카드 중 가장 가까운 카드 1장을 결과로 보여준다.

- **벡터 축 3개** (각 0~10):
  - **행동축**: 0=즉흥적 ↔ 10=계획적/체계적
  - **관계축**: 0=개인중심(혼자) ↔ 10=관계중심(사람과 함께)
  - **감정축**: 0=차분/안정 ↔ 10=예민/감성적
- 각 질문의 보기(A~D)마다 (행동, 관계, 감정) 값이 고정되어 있음(2번 표).
- 사용자가 6개 질문에 답하면, 선택한 6개 보기의 (행동, 관계, 감정) 값을 각각 평균 → 사용자 벡터 1개 산출.
- 22장 카드도 동일한 3축 좌표를 가지고 있음(3번 표).
- 사용자 벡터와 22장 카드 좌표 사이의 거리(맨해튼 거리: |행동차| + |관계차| + |감정차|)를 계산해 **가장 거리가 짧은 카드 1장**을 결과로 매칭.
- **동률 처리(타이브레이크)**: 거리가 같은 카드가 여러 장이면, 카드 번호(0 광대 → XXI 세계) 오름차순으로 가장 앞선 카드를 선택. 고정 규칙이라 매번 같은 결과 보장.
- 질문/보기/카드 좌표가 모두 고정값이라 AI 호출이나 난수 없이 결정론적으로 동작 — 코딩 단계에서 그대로 구현 가능.

## 4. 결과 화면 — 카드별 "당신을 닮은 카드" 문구

각 문구는 운세 해석 톤과 달리 **성향/캐릭터 소개 톤**(MBTI 결과 화면과 비슷한 느낌)으로 작성. 결과 화면에 문구 + `/tarot/[card-name]` 상세 링크 + "다시 테스트하기"/"친구에게 공유하기" 버튼을 함께 노출.

| 카드 | 결과 문구 |
|---|---|
| 0 광대 | 당신은 0번, 광대(The Fool) 카드를 닮았어요! 정해진 길보다 마음 가는 대로 움직이는 자유로운 모험가 타입이에요. 즉흥적인 선택이 오히려 새로운 기회를 열어주는 경우가 많아요. |
| I 마법사 | 당신은 I번, 마법사(The Magician) 카드를 닮았어요! 원하는 걸 현실로 만들어내는 능력자 타입이에요. 목표가 생기면 바로 행동으로 옮기는 추진력이 강점이에요. |
| II 여사제 | 당신은 II번, 여사제(The High Priestess) 카드를 닮았어요! 말보다 분위기와 직감으로 상황을 읽는 타입이에요. 조용하지만 누구보다 깊이 있는 통찰력을 가졌어요. |
| III 여황제 | 당신은 III번, 여황제(The Empress) 카드를 닮았어요! 주변 사람을 따뜻하게 품어주는 타입이에요. 풍요롭고 안정적인 분위기를 자연스럽게 만들어내요. |
| IV 황제 | 당신은 IV번, 황제(The Emperor) 카드를 닮았어요! 계획을 세우고 이끄는 데 능숙한 타입이에요. 책임감이 강해서 주변에서 자연스럽게 의지하게 돼요. |
| V 교황 | 당신은 V번, 교황(The Hierophant) 카드를 닮았어요! 원칙과 신뢰를 중요하게 여기는 타입이에요. 안정적인 기준을 세워두고 차분히 따라가는 걸 좋아해요. |
| VI 연인 | 당신은 VI번, 연인(The Lovers) 카드를 닮았어요! 사람과의 케미와 연결을 가장 소중히 여기는 타입이에요. 마음이 맞는 사람과는 금방 가까워져요. |
| VII 전차 | 당신은 VII번, 전차(The Chariot) 카드를 닮았어요! 목표가 생기면 망설임 없이 돌진하는 타입이에요. 추진력과 속도감 있는 결단력이 매력이에요. |
| VIII 힘 | 당신은 VIII번, 힘(Strength) 카드를 닮았어요! 부드럽지만 단단한 내면을 가진 타입이에요. 힘든 순간에도 묵묵히 버텨내는 힘이 있어요. |
| IX 은둔자 | 당신은 IX번, 은둔자(The Hermit) 카드를 닮았어요! 혼자만의 시간 속에서 답을 찾는 타입이에요. 깊이 있는 생각과 차분한 통찰이 강점이에요. |
| X 운명의 수레바퀴 | 당신은 X번, 운명의 수레바퀴(Wheel of Fortune) 카드를 닮았어요! 흐름에 몸을 맡기고 가볍게 받아들이는 타입이에요. 예상 밖의 변화도 기회로 잘 살려요. |
| XI 정의 | 당신은 XI번, 정의(Justice) 카드를 닮았어요! 공정함과 논리를 중요하게 여기는 타입이에요. 차분하게 따져보고 판단하는 데 능숙해요. |
| XII 매달린 사람 | 당신은 XII번, 매달린 사람(The Hanged Man) 카드를 닮았어요! 남들과 다른 시각으로 상황을 바라보는 타입이에요. 잠깐 멈춰서 생각하는 여유가 새로운 답을 찾아줘요. |
| XIII 죽음 | 당신은 XIII번, 죽음(Death) 카드를 닮았어요! 미련 없이 정리하고 새로 시작하는 타입이에요. 변화를 두려워하지 않는 결단력이 있어요. |
| XIV 절제 | 당신은 XIV번, 절제(Temperance) 카드를 닮았어요! 무엇이든 균형 있게 맞춰가는 타입이에요. 서두르지 않고 차근차근 조율하는 데 능숙해요. |
| XV 악마 | 당신은 XV번, 악마(The Devil) 카드를 닮았어요! 솔직한 욕망과 매력을 숨기지 않는 타입이에요. 가끔은 유혹에 흔들려도 그게 곧 당신만의 매력이에요. |
| XVI 탑 | 당신은 XVI번, 탑(The Tower) 카드를 닮았어요! 큰 변화 속에서도 결국 다시 일어서는 타입이에요. 무너진 자리에서 더 단단해지는 회복력이 있어요. |
| XVII 별 | 당신은 XVII번, 별(The Star) 카드를 닮았어요! 희망을 잃지 않고 묵묵히 나아가는 타입이에요. 주변 사람들에게 잔잔한 위로를 주는 존재예요. |
| XVIII 달 | 당신은 XVIII번, 달(The Moon) 카드를 닮았어요! 감정에 솔직하고 섬세한 타입이에요. 남들이 못 보는 미묘한 감정의 결을 잘 캐치해요. |
| XIX 태양 | 당신은 XIX번, 태양(The Sun) 카드를 닮았어요! 밝은 에너지로 주변을 환하게 만드는 타입이에요. 긍정적인 분위기 메이커 포지션이 잘 어울려요. |
| XX 심판 | 당신은 XX번, 심판(Judgement) 카드를 닮았어요! 끊임없이 한 단계씩 성장하려는 타입이에요. 지난 경험을 발판 삼아 더 나아지는 힘이 있어요. |
| XXI 세계 | 당신은 XXI번, 세계(The World) 카드를 닮았어요! 시작한 일은 끝까지 완성해내는 타입이에요. 무엇을 하든 완결감 있게 마무리하는 게 강점이에요. |
