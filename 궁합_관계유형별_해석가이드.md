# 궁합 모드 — 관계유형별 해석 가이드

`관계궁합_확장_기획.md` 1번에서 정의된 4종(연인/베프/상사/짝사랑) 궁합의 해석 텍스트를 만들 때 쓰는 어휘 매핑표 + AI 프롬프트 가드레일 + 짝사랑궁합 전용 안전장치. 산출 로직과 점수 구간은 `운세텍스트_생성방식.md`(0~20/21~40/41~60/61~80/81~100)와 동일하게 재사용하고, 이 문서는 그 위에 얹는 "유형별 어조 변환 규칙"만 다룬다.

## 1. 공통 원칙

- 4종 모두 종합 톤 가이드(`운세텍스트_생성방식.md` 2번)는 그대로 따른다: 친근한 존댓말, 단정적/공포 조성 문구 금지, 트렌디 표현은 가끔만.
- 연인궁합은 `운세_폴백_템플릿.md`의 애정 카테고리 문구를 그대로 재사용한다(가장 자연스러운 매칭).
- 베프궁합·상사궁합·짝사랑궁합은 애정/직장 폴백 문구를 가져와 아래 매핑표·가드레일을 거쳐 변환한 뒤 사용한다.

## 2. 베프궁합 — 어휘 매핑표 (연애 표현 → 케미/우정 표현)

베프궁합은 애정 가중치 컬럼을 재사용하지만 "연애" 어휘가 그대로 노출되면 안 되므로, AI 프롬프트와 폴백 템플릿 가공 시 아래 매핑을 적용한다.

| 연애 표현 (원본, 애정 카테고리) | 베프궁합 대체 표현 |
|---|---|
| 설렘 / 설레는 | 케미 / 통하는 느낌 |
| 연애운 | 우정운 / 케미 지수 |
| 연인 | 친구 |
| 솔로 | (해당 없음 — 문장 자체 제외) |
| 애정 | 우정 / 케미 |
| 호감 가는 사람과의 인연 | 잘 통하는 친구와의 케미 |
| 마음을 표현하면 | 솔직하게 얘기하면 / 먼저 다가가면 |
| 사랑 / 깊은 애정 | 끈끈한 우정 / 진한 케미 |
| 다정한 분위기 | 편안한 분위기 |
| 투닥거리다 (연인 간) | 장난스럽게 다투다 |
| 인생 인연 | 평생 갈 친구 |

**적용 예시** (애정 81~100 구간 원문 → 베프궁합 변환):
- 원문: "솔로라면 인생 인연을 만날 수도 있는 기대되는 하루예요." → 베프궁합: 해당 문장은 "솔로" 전제라 제외하고 다른 문구로 대체.
- 원문: "연인과는 깊은 애정을 확인할 수 있는 특별한 순간이 생길 수 있어요." → 베프궁합: "친구와는 끈끈한 우정을 확인할 수 있는 특별한 순간이 생길 수 있어요."
- 원문: "이 카드 좀 소름… 싶을 만큼 마음이 잘 통하는 날이 될 수 있어요." → 베프궁합: 그대로 사용 가능(연애 표현 없음).

**AI 프롬프트 가드레일 (베프궁합 전용 추가 문장)**:
> "이 결과는 친구 사이의 케미/우정 궁합입니다. '연애', '설렘', '연인', '사랑' 등 연애를 전제하는 단어는 쓰지 말고 '케미', '우정', '잘 통함', '편안함' 등의 표현으로 바꿔서 답변하세요."

## 3. 상사궁합 — 톤 가드레일

상사궁합은 직장 가중치 컬럼을 재사용한다. `운세_폴백_템플릿.md`의 직장 카테고리 문구는 대부분 평가적 어조가 아니라 그대로 쓸 수 있지만, "노력한 부분을 인정받을 수 있는" 류의 문장은 화자 기준이 모호해질 수 있어 주의가 필요하다.

- 금지: 상대(상사)를 일방적으로 평가하는 표현(예: "이 분은 일을 못해요", "상사로서 별로예요").
- 권장 대체 표현: "이 분과는 같이 일하기 좋은 흐름이에요", "소통이 원활하게 풀리는 날이에요", "보고나 협업 타이밍을 잡기 좋은 시기예요".

**AI 프롬프트 가드레일 (상사궁합 전용 추가 문장)**:
> "이 결과는 상사와 부하 직원 사이의 업무 궁합입니다. 상사 개인의 인성이나 능력을 평가하는 표현은 쓰지 말고, '협업이 잘 풀리는 흐름인지'에만 초점을 맞춰 격식 있는 존댓말로 답변하세요."

**적용 예시** (직장 61~80 구간 원문 → 상사궁합 변환):
- 원문: "노력한 부분을 인정받을 수 있는 기분 좋은 하루예요." → 상사궁합: "노력한 부분이 잘 전달되고, 협업이 매끄럽게 풀리는 하루예요."
- 원문: "동료나 상사와의 협업도 매끄럽게 흘러가는 날이에요." → 상사궁합: 그대로 사용 가능.

## 4. 짝사랑궁합 — 전용 안전장치 (가장 민감한 유형)

`관계궁합_확장_기획.md` 4번에서 예고된 안전장치를 구체화한다. 짝사랑궁합은 상대방의 마음을 결과로 단정하면 사용자에게 직접적인 상처를 줄 수 있는 민감한 콘텐츠이므로 별도 규칙을 둔다.

### 4.1 절대 금지 표현 (필터 차단 대상)
- 상대의 감정을 단정하는 모든 표현: "그 사람은 당신을 안 좋아해요", "짝사랑은 이루어지지 않아요", "상대는 관심이 없어요"
- 관계의 결말을 확정하는 표현: "결국 잘 안 될 거예요", "이 사람은 인연이 아니에요"
- 행동을 강요/단정하는 표현: "절대 다가가지 마세요", "지금 바로 고백하세요"

### 4.2 권장 대체 표현 (가능성 중심 어법)
- "지금 흐름은 ~한 편이에요" (확정이 아닌 흐름으로 표현)
- "조금 더 다가가 보면 좋을 시기예요" (행동을 제안하되 강요하지 않는 톤)
- "마음을 천천히 살펴보는 게 좋은 타이밍이에요"
- "지금 단계에서는 ~한 신호가 보여요, 다만 관계는 계속 변할 수 있으니 여유를 갖고 지켜봐도 좋아요"

### 4.3 점수 구간별 적용 예시 (애정 폴백 → 짝사랑궁합 변환)
- 0~20 원문: "사소한 오해가 생기기 쉬운 날, 말은 한 번 더 다정하게 골라보세요." → 짝사랑: "지금은 서두르기보다 천천히 다가가는 게 좋은 흐름이에요."
- 41~60 원문: "솔로라면 특별한 기회는 없어도 마음은 평온한 하루예요." → 짝사랑: "지금 단계에서는 큰 변화보다 마음을 편하게 가져가는 게 좋은 시기예요."
- 81~100 원문: "솔로라면 인생 인연을 만날 수도 있는 기대되는 하루예요." → 짝사랑: "조금 더 다가가 보면 좋은 신호를 받을 수도 있는, 기대해볼 만한 흐름이에요."

### 4.4 AI 프롬프트 가드레일 (짝사랑궁합 전용 추가 문장, 필수 삽입)
> "이 결과는 짝사랑 궁합입니다. 상대방의 마음이나 관계의 결말을 절대 단정하지 마세요. '지금 흐름은', '~한 시기예요', '다가가 보면 좋을 때예요' 같은 가능성 중심 표현만 사용하고, 사용자가 실망하거나 상처받을 수 있는 직접적 거절·확정 표현은 금지합니다."

### 4.5 고정 안내 문구 (결과 화면 추가 노출, 짝사랑궁합 전용)
> "이 결과는 참고용이며, 실제 상대방의 마음과는 다를 수 있어요. 관계는 결과보다 직접 나누는 대화로 만들어가는 게 가장 정확해요."

이 문구는 `운세텍스트_생성방식.md` 5번의 공통 고지("참고용 콘텐츠이며 실제 결정의 근거로 사용하지 마세요")에 더해 짝사랑궁합 결과 화면에만 추가로 노출한다.

## 5. 기획안 반영 필요 사항

- `관계궁합_확장_기획.md` 1번 표에 본 문서 링크 추가.
